这是一个ChatGPT创作的tampermonkey脚本. 其作用为在gululu.world(也就是骨站)中辅助收安价

具体功能:
- 自动检测某层楼中看起来像是安价或者加权(仅检测评论的评论)的评论/评论的评论(规则可以设置),并将其汇总为有序列表
- 将其余评论(含评论的评论)列出以方便检查(已安价过的人的昵称标为灰色)
- 有序列表的初始编号可调(这是为了方便同时在其他平台上收安价的人)
- 可以选择对同一个用户的多次安价做不同的处理(目前支持: 全部保留/保留最早/保留最晚/合并)

声明:
- 虽然使用了`GM_xmlhttpRequest`,但经本人检查只用于获取评论
- (合理使用的情况下)访问骨站API的次数与手动收安价相同,尽可能不对土豆造成过多的负担

可能的风险:
- 骨站页面结构/API接口变化导致不可用
